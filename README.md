unity3d Cookbook
================

install unity3d to your mac os x

Requirements
------------

### Platform

- MacOSX

### Cookbooks

- dmg

Attributes
----------

* `node['unity3d']['dmg']['source']` - Unity3D dmg URL
* `node['unity3d']['dmg']['checksum']` - its checksum (`shasum -a unity-x.y.z.dmg`)
